# Listas da Matéria - Programação de Computadores

Repositório contendo listas de exercícios de programação e suas resoluções.

## 📋 Estrutura do Repositório

```
lista_programacao_computadores/
├── listas/              # Enunciados das listas de exercícios
├── resolucoes/          # Soluções dos exercícios
│   └── lista01.ipynb
└── README.md
```

## 📚 Conteúdo

- **listas/**: Contém os enunciados e descrições dos exercícios propostos
- **resolucoes/**: Contém as resoluções dos exercícios em formato Jupyter Notebook

## 🚀 Como Usar

### Pré-requisitos

- Python 3.x
- Jupyter Notebook
- Bibliotecas necessárias (conforme especificado em cada resolução)

### Executando os Notebooks

1. Clone este repositório:
```bash
git clone https://github.com/seu-usuario/lista_programacao_computadores.git
cd lista_programacao_computadores
```

2. Instale as dependências (se houver `requirements.txt`):
```bash
pip install -r requirements.txt
```

3. Inicie o Jupyter Notebook:
```bash
jupyter notebook
```

4. Abra o arquivo desejado na pasta `resolucoes/`
